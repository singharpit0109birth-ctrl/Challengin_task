# Crop Profit Calculator

## üìå Overview

The **Crop Profit Calculator** is a simple Python-based tool designed to help farmers or agricultural planners estimate their potential profit or loss based on land size, crop yield, selling price, and plantation cost. By inputting basic agricultural parameters, the program calculates total revenue, total cost, and final profit.

---

## ‚≠ê Features

- Accepts user inputs for:
  - Land area (in acres)
  - Crop yield per acre (in quintals)
  - Selling price per quintal (in rupees)
  - Plantation cost per acre (in rupees)
- Automatically computes:
  - Total crop yield
  - Total cost of plantation
  - Total revenue
  - Final profit or loss
- Simple and easy to use for beginners
